Cron翻訳ツール テストページ

このページには様々なCron式の例が含まれています。拡張機能が有効になっていると、各Cron式にマウスオーバーすると日本語翻訳が表示されます。

基本的なCron式

0 0 * * *
毎日深夜0時に実行
*/5 * * * *
5分ごとに実行
0 */2 * * *
2時間ごとに実行
30 8 * * MON-FRI
平日（月曜日から金曜日）の朝8時30分に実行
0 0 * * 1-5
平日（月曜日から金曜日）の深夜0時に実行

複雑なCron式

0 9-17 * * 1-5
平日の9時から17時まで毎時0分に実行
15 10,14,18 * * *
毎日10:15、14:15、18:15に実行
0 0 1 */3 *
3ヶ月ごとの1日に実行
0 0 15 * *
毎月15日の深夜0時に実行

秒付きCron式（6フィールド）

0 */5 * * * *
5分ごとの0秒に実行
30 10 * * * *
毎時10分30秒に実行

スクリーンショット用サンプル

システムのバックアップスケジュール:

  • 日次バックアップ: 0 2 * * * （毎日午前2時）
  • 週次バックアップ: 0 3 * * 0 （毎週日曜日午前3時）
  • 月次バックアップ: 0 4 1 * * （毎月1日午前4時）
